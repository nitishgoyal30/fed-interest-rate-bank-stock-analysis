# Impact of Federal Reserve Interest Rate Changes on U.S. Bank Stock Returns (2005–2025)

## Author
Nitish Goyal

## Overview
This repository contains Python code, datasets, and regression results used in the empirical analysis of the impact of Federal Reserve interest rate changes on stock returns of major U.S. banks.

The study examines how monetary policy shocks influence bank equity performance using econometric and financial data analysis techniques.

## Methodology
- Data collection from Federal Reserve and financial markets
- Data cleaning and preprocessing using Python (Pandas, NumPy)
- Regression analysis using Statsmodels
- Visualization using Matplotlib
- Empirical analysis of bank stock sensitivity to interest rate changes

## Files included
- bank_interest_rate_research-2.ipynb → Main Python analysis notebook
- final_dataset.csv → Final dataset used for analysis
- bank_interest_rate_dataset.csv → Raw dataset
- regression_results.txt → Regression outputs
- .png files → Graphs and visualizations
